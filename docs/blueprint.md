# **App Name**: LinguaLive

## Core Features:

- Speech Analysis & Correction: AI-powered analysis of user speech to identify grammatical errors, incorrect vocabulary, and unnatural phrasing. It leverages a reasoning tool to prioritize key errors and phrasing improvements to provide more focused suggestions.
- Dynamic Response Generation: Generation of real-time responses that are grammatically correct and incorporate natural-sounding language, using a tool to vary sentence structure, idiomatic expressions and vocabulary.
- Real-Time Text Display: Clear and concise display of the AI avatar's spoken responses in text form.
- Speech-to-Text Input: Capture user's speech via microphone and transcribe it into text for processing.
- Interactive Chat Interface: Simple, intuitive interface for real-time conversation, inspired by modern messaging apps.

## Style Guidelines:

- Primary color: Soft blue (#75A9FF) to create a calming and trustworthy atmosphere.
- Background color: Very light blue (#F0F8FF), almost white, for a clean and approachable feel.
- Accent color: Light green (#90EE90) to highlight positive feedback and encourage improvement.
- Clear and readable sans-serif fonts for ease of reading, even in real-time.
- Clean layout with distinct sections for user input, AI responses, and feedback, promoting clarity and reducing cognitive load.
- Simple and friendly icons to represent key features, such as a microphone for speech input and a lightbulb for tips.
